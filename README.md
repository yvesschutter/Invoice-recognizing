# Invoice-recognizing

This project contains a Tkinter application for recognizing invoice data in PDF files.

## Setup

Install the required packages:

```bash
pip install -r requirements.txt
```

The application expects a `config.json` file in the same directory as the script. Adjust the paths in this file to your environment.

## Usage

Run the application with Python:

```bash
python "Factuur_app23 -test.py"
```
